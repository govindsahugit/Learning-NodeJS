## â˜ï¸ What Is Cloud Computing?

**Cloud computing** means performing computation on **remote machines** instead of local ones.

The nature of computing does not change:

* computations still happen
* binary operations still occur
* CPU and memory are still used

What changes is:

* where the computation runs
* who owns and manages the machines

In cloud computing, you use computing power provided by cloud providers over the internet, instead of maintaining your own machines. We are also using cloud compute by hosting our storage app on EC2 instance.
